# Reproductor de Música Simple

## Características Principales

### Interfaz de Usuario
- Botón para cargar archivos de audio
- Controles de reproducción/pausa
- Barra de progreso
- Control de volumen
- Lista de reproducción
- Tiempo transcurrido y duración total

### Funcionalidades
- Carga múltiples archivos de audio
- Reproducción/Pausa
- Navegación en la línea de tiempo de la canción
- Ajuste de volumen
- Reproducción automática de la siguiente canción
- Selección de canciones desde la playlist
- Visualización de tiempo transcurrido y total

## Cómo Usar

1. Haz clic en el botón "Seleccionar archivos" (input tipo file)
2. Selecciona uno o más archivos de audio desde tu dispositivo
3. Las canciones se agregarán automáticamente a la lista de reproducción
4. Haz clic en cualquier canción de la lista para reproducirla
5. Utiliza los controles para:
   - Pausar/reanudar la reproducción
   - Ajustar el volumen
   - Saltar a diferentes partes de la canción

## Notas
- Soporta formatos de audio compatibles con el navegador (MP3, WAV, OGG)
- Personalizable modificando las clases CSS en el archivo de estilos
